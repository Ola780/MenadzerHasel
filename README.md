Program ten będzie przechowywać hasła wraz z źródłami, które zostaną udostępnione użytkownikowi po wcześniejszym wpisaniu głównego hasła.Użytkownik po uruchomieniu programu powinien mieć możliwość wyboru pliku, w którym zaszyfrowane są hasła lub utworzenie nowego pliku z hasłami. Po wyborze pliku powinien zostać poproszony o podanie hasła, które posłuży do odszyfrowania pliku. Niezależnie od tego, czy wpisane przez użytkownika hasło jest poprawne czy nie, program powinien wyświetlić dane i umożliwić pracę na pliku (oczywiście, przy podaniu nieprawidłowego hasła wyświetlone dane również będą nieprawidłowe).
